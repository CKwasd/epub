分手之后马上成为了义理姐弟的恋人之间的故事——紙城鏡介

“毫无疑问我是你哥啊。”

“我当然是你姐姐啊。”

“我讨厌的就是你这样的地方啊，你个死推理狂。”

“我讨厌的就是你这样的地方啊，你个死宅。”
